题目描述

将输入的字符串按特定规则进行Z字形变换后输出。

具体来说，给定一个字符串 s 和一个行数 numRows，将 s 以从上到下、从左到右进行 Z 字形排列。比如输入字符串为 “PAYPALISHIRING” 行数为 3 时，排列如下：

P   A   H   N

A P L S I I G

Y   I   R

之后按从左往右、从上往下的顺序依次输出所有字符即可得到答案：”PAHNAPLSIIGYIR”。

请你实现一个将字符串进行指定行数变换的函数

fn convert(s: String, num_rows: i32) -> String {
}

思路

本题中，要求将输入的字符串按一定规则进行 Z 字形变换，并输出转换后的字符串。具体而言，就是将原始字符串从上到下、从左到右排列成一个“之”字形状。

为了完成这个任务，我们需要选择一个合适的数据结构来存储变换后的字符串。常用的数据结构有二维数组、向量等。在本题中，我们可以使用一个向量数组来存储每一行中的字符。对于一个长度为 n 的字符串，我们需要尽可能地将其中的字符依次填入向量数组中。具体来说，我们可以按照以下规则进行顺序填充：

从上到下一行一行地填充，直到达到指定行数 numRows。

在到达最后一行时，反方向执行相同的操作，从下往上一行一行地填充，直到回到第一行。

重复执行步骤 1 和 2，直到字符串中的所有字符都被填充完毕。

遍历完成后，我们就可以得到一个按 Z 字形排列的向量数组。将该数组按行连接起来即可得到转换后的字符串。

根据上述思路，我们可以完成如下代码实现，并给出自测试用例。

代码实现

fn convert(s: String, num_rows: i32) -> String {
    // 首先进行边界判断
    if num_rows <= 1 {
        return s;
    }

    let mut rows = Vec::new();
    // 向量数组每个元素表示每一行的字符
    for _ in 0..num_rows {
        rows.push(Vec::new());
    }

    let mut cur_row = 0i32;
    let mut going_down = false;

    // 遍历字符串，依次填充向量数组中的元素
    for c in s.chars() {
        rows[cur_row as usize].push(c);
        // 如果当前在第一行或最后一行，需要切换方向
        if cur_row == 0 || cur_row == num_rows - 1 {
            going_down = !going_down;
        }
        // 根据方向决定下一行的位置
        cur_row += if going_down { 1 } else { -1 };
    }

    let mut ret = String::new();
    for row in rows {
        ret.push_str(&row.iter().collect::<String>());
    }
    ret
}

测试用例

fn main() {
    // 测试用例
    assert_eq!(convert("PAYPALISHIRING".to_string(), 3), "PAHNAPLSIIGYIR");
}

总结

本题中，我们学习了如何使用向量数组以及遍历字符串的方法，实现了一个可以将输入字符串按指定规则进行 Z 字形变换的函数。同时，我们注意到在处理边界情况时需要特别小心，这有助于我们编写健壮的代码
