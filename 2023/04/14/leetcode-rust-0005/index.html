「最长回文子串」

给定一个字符串 s，找到 s 中最长的回文子串。你可以假设 s 的最大长度为 1000。

回文串指的是从左往右读和从右往左读完全相同的字符串。例如，”aba” 和 “abba” 都是回文串，而 “abc” 不是。

思路

为了解决这个问题，我们可以使用中心扩展算法。对于每个位置 i，我们可以分别向左和向右扩展，直到无法扩展为止。注意，会有两种情况需要考虑：

回文串长度为奇数，此时以 i 为中心；
回文串长度为偶数，此时以 i 和 i+1 为中心。
通过遍历所有可能的中心点，并更新最长回文子串，即可求出答案。

以下是 Rust 语言实现的代码和注释：

代码实现

fn longest_palindrome(s: String) -> String {
    let n = s.len();
    if n < 2 {
        return s;
    }

    let s_chars: Vec<char> = s.chars().collect(); // 将字符串转换为字符数组
    let mut start = 0; // 记录最长回文子串的起始下标
    let mut end = 0; // 记录最长回文子串的结束下标

    for i in 0..n {
        let len1 = expand_around_center(&s_chars, i, i); // 以 i 为中心的回文串长度
        let len2 = expand_around_center(&s_chars, i, i + 1); // 以 i 和 i+1 为中心的回文串长度

        let len = usize::max(len1, len2); // 取两种情况中回文串长度的最大值
        if len > end - start {
            start = i - (len - 1) / 2;
            end = i + len / 2;
        }
    }

    s[start..=end].to_string() // 返回最长回文子串
}

fn expand_around_center(chars: &Vec<char>, left: usize, right: usize) -> usize {
    let mut l = left as i32;
    let mut r = right as i32;

    while l >= 0 && r < chars.len() as i32 && chars[l as usize] == chars[r as usize] {
        l -= 1;
        r += 1;
    }

    (r - l - 1) as usize // 返回回文串的长度
}

测试用例

#[test]
fn test_longest_palindrome() {
    assert_eq!(longest_palindrome("babad".to_string()), "bab".to_string());
    assert_eq!(longest_palindrome("cbbd".to_string()), "bb".to_string());
    assert_eq!(longest_palindrome("a".to_string()), "a".to_string());
    assert_eq!(longest_palindrome("ac".to_string()), "a".to_string());
    assert_eq!(longest_palindrome("abb".to_string()), "bb".to_string());
}
