无重复字符的最长子串

题目描述

给定一个字符串，请你找出其中不含有重复字符的最长子串的长度。

示例 1:

输入: “abcabcbb”

输出: 3

解释: 因为无重复字符的最长子串是 “abc”，所以其长度为 3。

示例 2:

输入: “bbbbb”

输出: 1

解释: 因为无重复字符的最长子串是 “b”，所以其长度为 1。

示例 3:

输入: “pwwkew”

输出: 3

解释: 因为无重复字符的最长子串是 “wke”，所以其长度为 3。

请注意，答案必须是一个子串，”pwke” 是 子序列 而不是子串。

思路

这道题可以使用滑动窗口进行求解。我们维护一个左指针 left 和一个右指针 right，表示当前考虑的无重复子串的左右边界。

具体来说，每次移动右指针，则扩大了当前考虑的子串的范围，直到遇到重复字符为止。此时，我们就找到了以 right-1 为结尾的，最长的无重复子串。在此过程中，我们可以记录下最长的子串的长度并不断更新。

接着，我们将左指针向右移动一位，并重复上述过程，直到右指针到达字符串末尾为止。最后，我们就能够找出整个字符串中的最长无重复子串。

代码实现

fn length_of_longest_substring(s: String) -> i32 {
    let mut left = 0;
    let mut right = 0;
    let mut max_len = 0;
    let mut char_set = std::collections::HashSet::new();
    let chars: Vec<char> = s.chars().collect();

    while right < chars.len() {
        if !char_set.contains(&chars[right]) {
            // 如果当前字符不在集合中，则将其加入集合，并更新最长子串长度
            char_set.insert(chars[right]);
            right += 1;
            max_len = max_len.max(right - left);
        } else {
            // 如果当前字符已经在集合中，则需要将左指针向右移动，并从集合中删除相应字符
            char_set.remove(&chars[left]);
            left += 1;
        }
    }
    max_len as i32
}

测试用例

#[test]
fn test_length_of_longest_substring() {
    assert_eq!(length_of_longest_substring("abcabcbb".to_string()), 3);
    assert_eq!(length_of_longest_substring("bbbbb".to_string()), 1);
    assert_eq!(length_of_longest_substring("pwwkew".to_string()), 3);
    assert_eq!(length_of_longest_substring("".to_string()), 0);
    assert_eq!(length_of_longest_substring("a".to_string()), 1);
}
